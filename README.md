# Currency Converter App (Vite + React)

This project is a simple application designed to compare and convert currency rates. It is built using Vite and React, and allows currency conversion based on real-time exchange rates for TRY, EUR, and USD.

# Installation:
1- Clone this repository to your local machine:
git clone https:// github.com /your-username /currency-converter-app.git


2- Install the required dependencies:
npm install


3- Start the application:
npm run dev
